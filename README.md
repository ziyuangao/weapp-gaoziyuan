## 📋 项目简介

**小原的百宝箱** 是一个纯粹的个人工具箱小程序。初衷是做一个干干净净的小工具合集，方便自己和身边人日常使用。

这里**没有登录、没有广告、不会收集任何个人信息**。所有功能的数据均来源于互联网上可靠的第三方公开接口，我只做筛选、整合与清晰呈现。

> “如果这个小工具偶尔帮到了你，我会很开心。”
